## Detection Signals

### Signal 1: LLM Classification via Groq
Prompt the Groq API (Llama 3) with the submitted text. Ask the model to return a probability between 0.0 and 1.0 that the content was AI generated. Parse the float from the response. If parsing fails, fall back to 0.5 (uncertain).

### Signal 2: Stylometric Heuristics
Extract these features from the raw text:
- Average sentence length (word count per sentence)
- Standard deviation of sentence length
- Type to token ratio (unique words / total words)
- Comma count per 100 words
- Count of common AI hedging phrases

Normalize each feature to a 0 to 1 scale against known baselines. Average them into a single stylometric score.

---

## Confidence Scoring Design

```
final_score = (groq_score * 0.65) + (stylo_score * 0.35)
```

Thresholds:
- Score >= 0.75 -> AI Generated (High Confidence)
- Score <= 0.30 -> Human Authored (High Confidence)
- Score between 0.31 and 0.74 -> Uncertain Origin

---

## Transparency Label Design

Each label has three fields: a short name, a score range, and a plain language message shown to the user.

| Label | Score Range | User Message |
|---|---|---|
| AI Generated (High Confidence) | >= 0.75 | "Our system found strong indicators that this content was generated by an AI tool." |
| Human Authored (High Confidence) | <= 0.30 | "Our system found strong indicators that this content was written by a human." |
| Uncertain Origin | 0.31 to 0.74 | "Our system could not determine with confidence the origin of this content. You may submit an appeal." |

---

## Appeals Workflow

1. Creator submits POST /appeal with submission_id and reason.
2. System validates that submission_id exists in audit log.
3. Appeal is logged as a new entry with appeal_id, timestamp, reason, and status set to under_review.
4. Audit log entry for the original submission is updated with appeal_status = under_review.
5. Creator receives confirmation response.
6. Human review step is manual and outside the scope of this implementation.

---

## Edge Cases

- Text under 50 words: stylometric score is flagged as low confidence; final score is weighted 90% toward LLM signal.
- Non English text: stylometric signal is skipped; only LLM signal is used.
- Groq API timeout or error: LLM score defaults to 0.5; label is forced to Uncertain Origin regardless of stylometric score.
- Duplicate submission_id on appeal: return 409 Conflict.
- Empty content field: return 400 Bad Request with validation message.
